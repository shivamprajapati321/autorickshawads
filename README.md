# AutoRickshawAds

Auto Rickshaw Advertising website — Hood Advertising, Back Panel Advertising & Hood Manufacturing across 200+ cities.

## Structure
- `index.html` — Homepage
- `assets/` — Rickshaw icons (yellow / white / black PNG)
- `robots.txt` — Crawl rules
- `sitemap.xml` — SEO sitemap

## Deployment
Static site. Hosted on Vercel (auto-deploy on push to `main`).
Domain: autorickshawads.com (DNS via Hostinger → Vercel).

## Notes
- No build step required (pure HTML/CSS/JS).
- Framework preset on Vercel: **Other**.
